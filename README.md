# KeepMeOnTrack

KeepMeOnTrack is a minimalist Progressive Web App (PWA) that helps you build one habit at a time in the simplest way possible: don't let your daily chain break.

> "One habit. One chain. Don't break it."

## Features
- **Target Setup**: Set up your habit target easily without an account.
- **Calendar Chain View**: Visualize your habit chain on a calendar grid.
- **Progress View**: Track your overall progress, milestones, and streaks.
- **Shareable Completion Card**: Celebrate and share your success when you complete your target.
- **Privacy First**: All data is stored locally in your browser's `localStorage`. No backend, no accounts.
- **PWA Capabilities**: Installable, offline-first, and fully responsive.

## Tech Stack
- Frontend: React + Vite
- Styling: Tailwind CSS
- State Management: Zustand
- Animations: Framer Motion
- Storage: `localStorage`

## Local Development
To run this project locally, ensure you have Node.js installed, then execute:

```bash
npm install
npm run dev
```
